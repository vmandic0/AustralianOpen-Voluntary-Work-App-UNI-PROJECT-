# AustralianOpen-Voluntary-Work-App-UNI-PROJECT-
University project in Java with mysql database, Client Server architecture
What is app purpose

Its Client Server project that alows users to submit for work either with an account or as guest, the put info in the UI and that info is sent to server and server that saves that info into the database adn sends users the back information.
UI i mostly written in serbian laguage!
Everything is done fore localhost because this is the uni project!

Projekat za prijavu volontera za Australian Open
Projekat se sastoji iz dve aplikacije, klijentske i serverske
Klijentska omogućava korisniku unos podataka dok serverska vrši obradu tih podataka
